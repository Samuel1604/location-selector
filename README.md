# Location Interaction Tracker (Vanilla JavaScript)
Location Interaction Tracker is a JavaScript based web application that allows users to explore and select different locations based on categories such as productive or fun.

The system tracks user interactions in real time, recording how often each location is selected and dynamically updating a history list. Frequently selected locations are prioritized, giving insight into user behavior patterns.

Key features include:
- Dynamic filtering of locations by category
- Real-time tracking of user selections
- Automatic count updates for repeated interactions
- History management with removal functionality
- Clean and responsive user interface

This project demonstrates strong understanding of JavaScript fundamentals, including array manipulation, event handling, and DOM updates, while simulating a simple user behavior tracking system.
